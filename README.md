# vue-cordova-car-control

## Project setup
```
yarn install
yarn run cordova-prepare
yarn run cordova-serve-android
cordova run android
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
